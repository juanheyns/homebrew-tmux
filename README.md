# homebrew-tmux (deprecated)

This tap has been consolidated into [`juanheyns/homebrew-tap`](https://github.com/juanheyns/homebrew-tap).

## Migrate

```bash
brew uninstall atmux
brew untap juanheyns/tmux
brew install juanheyns/tap/atmux
```

New releases of `atmux` will publish to `juanheyns/homebrew-tap` only.
